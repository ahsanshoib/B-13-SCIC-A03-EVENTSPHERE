🎉 EventSphere

EventSphere is a modern event booking platform built with TypeScript, designed to help users discover, explore, and book events through a clean, responsive, and user-friendly experience.

🌐 Live Site : https://b-13-scic-a03-eventsphere.vercel.app

🛠️ Tech Stack

🎨 Frontend

- React.js / Next.js
- TypeScript
- Tailwind CSS
- Recharts / Chart.js

🔐 Authentication

- Firebase Authentication

📦 Data Management

- Static JSON Data

🔐 Authentication System

EventSphere uses Firebase Authentication to provide secure user authentication and protected routes.

Supported features include:

- Email & Password Registration
- Login Validation
- Error Handling
- Demo Login Support
- Route Protection

---

📌 Project Architecture

«This project intentionally uses Static Data alongside Firebase Authentication instead of a dedicated backend server.»

This lightweight architecture allows the application to maintain excellent performance while demonstrating frontend architecture, authentication flow, protected routes, and professional UI/UX practices.

---

🔑 Environment Variables

Create a ".env.local" file and add your Firebase credentials:

NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
